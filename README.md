# Sticker Splitter

한 장의 콜라주 이미지를 넣으면 내부의 개별 오브젝트를 자동으로 분리해서 스티커처럼 잘라내는 간단한 브라우저 앱입니다.

## 실행 방법

1. [index.html](C:\Users\cjh86\Documents\Codex\2026-04-22-files-mentioned-by-the-user-chatgpt-2\index.html)을 브라우저에서 엽니다.
2. 이미지를 업로드합니다.
3. `스티커 분리하기` 버튼을 누릅니다.
4. 필요하면 민감도, 최소 크기, 테두리 두께를 조정한 뒤 다시 실행합니다.
5. 분리된 스티커를 개별 PNG로 저장하거나 `전체 다운로드`를 눌러 순차 저장합니다.

## 현재 버전 특징

- 설치 없이 브라우저만으로 실행 가능
- 밝은 배경의 콜라주 이미지에서 개별 오브젝트 자동 분리
- 흰색 스티커 테두리와 그림자 효과 자동 생성
- 분리 결과 미리보기와 PNG 저장 지원

## 잘 맞는 입력 이미지

- 배경이 흰색 또는 단색에 가까운 이미지
- 오브젝트끼리 충분히 떨어져 있는 이미지
- 투명 PNG 또는 배경과 요소 색 차이가 있는 이미지

## 한계

- 서로 겹치거나 닿아 있는 요소는 하나의 스티커로 합쳐질 수 있습니다.
- 배경과 매우 비슷한 밝은 요소는 민감도 조정이 필요할 수 있습니다.
- 복잡한 사진 배경에는 맞지 않고, 샘플처럼 스티커 시트나 콜라주에 더 적합합니다.
