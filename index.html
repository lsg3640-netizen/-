Sticker Splitter

한 장의 이미지에서 낱개 스티커를 자동으로 분리해 보세요

배경이 밝고 오브젝트가 서로 떨어져 있는 콜라주 이미지를 넣으면, 개별 오브젝트를 잘라내고 흰색 스티커 테두리를 붙여 PNG로 저장할 수 있습니다.

이미지를 업로드하면 준비됩니다.

수동 보정

번호 박스를 클릭해 선택

인식이 잘못된 번호 박스를 클릭한 뒤 박스 안쪽을 드래그하면 이동, 변이나 모서리를 드래그하면 크기 조절할 수 있습니다.

선택된 박스는 Delete 키를 누르면 삭제됩니다. 자동 재분리를 다시 실행하면 수동 보정은 현재 결과 기준으로 다시 계산됩니다.

잘 되는 이미지

  • 배경이 흰색 또는 단색에 가까운 이미지
  • 오브젝트끼리 서로 닿지 않는 콜라주
  • 각 요소 사이에 여백이 충분한 이미지

밝은 오브젝트가 많은 경우에는 배경 감지 민감도를 낮추고 다시 분리해 보세요.
